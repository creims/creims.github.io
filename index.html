Hex Viewer: Source  | Live

A simple hex viewer undertaken to learn ES6. Inspired by the hex editor Frhed.

Metronome: Source  | Live

A flexible metronome. Capable of playing complex polyrhythms.

A bass/guitar chord chart generator. Can handle arbitrary tunings as well as any number of strings.

RML Parser: Source

A parser/playback engine for an alpha-stage music notation language (think sheet music).

Chess Amateur: Source

A CLI chess game written in C. Inspired by the old ChessMaster software.
